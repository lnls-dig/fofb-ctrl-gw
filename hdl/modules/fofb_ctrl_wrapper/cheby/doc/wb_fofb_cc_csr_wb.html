- - - tfs_override cc_enable err_clr unused act_part
  • act_part [rw]: signals FOFB CC module to read configuration RAM
    write 1: trigger a new read to configuration RAM
  • unused [rw]: unused
    unused
  • err_clr [rw]: clears gigabit transceiver errors
    write 1: clears gigabit transceiver errors
  • cc_enable [rw]: enables CC module
    write 1: enables CC module
    write 0: disables CC module (acts as reset)
  • tfs_override [rw]: timeframe start override. BPM can override internal timeframe start signal and use MGT generated one.
    write 1: overrides timeframe start. BPM uses external signal.
    write 0: normal operation. BPM uses internal generated signal.
